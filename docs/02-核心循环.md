# 《开天辟地之大地的裂变》核心循环

## 1. 文档目的

本文档定义《开天辟地之大地的裂变》中玩家从进入游戏、获得角色、参与每轮行动，到最终飞跃创界山的完整体验循环。

本章重点回答以下问题：

1. 玩家进入游戏后首先做什么？
2. 一轮游戏如何开始和结束？
3. 上帝、世界牌、身份、种族、血牌和元宝如何共同参与游戏？
4. 黑夜行动、地图移动、战斗、交易和真心话大冒险如何衔接？
5. 玩家如何从普通角色成长为有资格挑战创界山的候选人？
6. 如何让几十种已有游戏的机制同时存在，却不会变成互不相关的小游戏合集？
7. 大地裂变如何打破长期积累，并将游戏推向终局？

核心循环必须优先保留陆展博在剧中讲述和演示的内容。

---

## 2. 原作核心流程提炼

### 2.1 剧中明确或高度可确认的内容

游戏中存在以下基本内容：

* 玩家共同参与一场大型桌游
* 玩家推选一人担任“上帝”
* 上帝负责主持、发牌、翻牌和指引部分秘密行动
* 玩家会获得身份牌、种族牌、血牌和元宝
* 玩家还会摸取基础游戏手牌
* 游戏存在骰子和棋盘移动
* 每回合开始时，上帝会进行判定
* 判定结果与翻开卡牌的花色有关
* 游戏存在黑夜
* 黑夜期间玩家闭眼，由上帝依次唤醒拥有不同身份或能力的角色
* 杀手、妖侠、法师、盗贼、魔王、神仙、长老等概念参与黑夜流程
* 地图移动过程中会触发特殊格子或局部规则
* 游戏中存在元宝
* 游戏中存在点卡
* 游戏中存在真心话大冒险、礼物交换或平安夜等聚会游戏机制
* 最终目标与创界山有关

现有公开整理通常将开局概括为：上帝发放身份牌、种族牌、血牌和元宝，其余游戏牌置于中央，玩家摸取初始手牌；每轮开始由上帝翻牌判定世界状态，黑桃会进入黑夜，并按特定身份顺序进行秘密行动。

### 2.2 原作候选规则说明

部分网络版本还整理了“二十面骰”“五个元宝发动真心话大冒险”“贿赂上帝”“购买点卡”“神鬼契约”等规则，但这些整理可能混合了剧中台词、画面道具和观众补全。

本项目暂时将它们视为“原作候选规则”，在后续逐条核对剧集后再决定是否升级为剧中明确规则。

## 3. 核心循环总览

一局游戏由五层循环共同组成。

```text
对局循环
└── 世界阶段循环
    └── 轮次循环
        └── 玩家行动循环
            └── 局部游戏循环
```

它们分别解决不同问题。

### 3.1 对局循环

描述整局游戏从创建世界到最终结算的过程。

```text
创建房间
→ 选择上帝与规则模块
→ 发放角色组件
→ 创建创界大陆
→ 开天阶段
→ 辟地与发展阶段
→ 裂变阶段
→ 创界山阶段
→ 最终结算
→ 生成战报
```

### 3.2 世界阶段循环

描述世界整体如何从稳定逐步走向裂变。

```text
开天
→ 辟地
→ 争界
→ 裂变
→ 创界
```

### 3.3 轮次循环

描述上帝如何通过世界牌推动一轮游戏。

```text
上帝判定
→ 世界状态生效
→ 秘密行动
→ 公开行动
→ 区域与局部游戏
→ 世界结算
→ 裂变检查
```

### 3.4 玩家行动循环

描述玩家轮到自己时做出的基本决策。

```text
观察局势
→ 选择移动或停留
→ 进入区域
→ 执行主要行动
→ 触发卡牌或事件
→ 与玩家或 NPC 互动
→ 获得或失去资源
→ 调整下一步计划
```

### 3.5 局部游戏循环

描述几十种已有游戏如何作为模块进入主世界。

```text
进入特定区域或触发事件
→ 加载局部规则
→ 支付参与条件
→ 完成短局玩法
→ 产生明确结果
→ 将奖励与后果带回主世界
```

局部游戏不能成为与主线无关的独立小游戏。

它必须改变至少一项主世界状态，例如：

* 生命
* 手牌
* 元宝
* 点卡
* 装备
* 身份信息
* 区域控制权
* 玩家关系
* 任务进度
* 裂变值
* 创界山资格

## 4. 游戏开始循环

### 4.1 创建房间

房主创建联机房间，并设定：

* 玩家人数
* 经典模式或自定义模式（一般都是经典即可）
* 扩展模块
* 回合计时
* 裂变速度
* 创界山终局类型
* 是否启用真人上帝

经典模式必须保留剧中核心模块：

* 上帝
* 身份牌
* 种族牌
* 血牌
* 元宝
* 世界判定
* 黑夜行动
* 地图移动
* 点卡
* 真心话大冒险
* 平安夜
* 创界山

### 4.2 推选上帝

玩家之间推选一人担任上帝。

数字版经典模式默认由系统担任上帝，以保证：

* 随机结果可信
* 隐藏信息不会泄露
* 所有玩家都可以正常参赛
* 规则能够自动判定
* 掉线不会导致主持流程中断

界面和规则中仍然使用“上帝”这一名称。

自定义模式可以允许真人上帝存在。

### 4.3 发放角色组件

上帝向每名玩家发放：

* 身份牌
* 种族牌
* 血牌
* 初始元宝
* 初始手牌

为了支持完整角色成长，数字版还会生成：

* 初始职业
* 基础属性
* 出生位置
* 个人目标
* 初始装备或道具
* 可用夜间行动
* 所属阵营或潜在阵营

角色信息分为三类。

公开信息：

* 角色名称
* 公开职业
* 当前生命状态
* 所在位置
* 已公开装备
* 公开称号

私有信息：

* 手牌
* 隐藏身份
* 个人目标
* 秘密任务
* 部分技能
* 夜间行动结果

条件公开信息：

* 种族
* 阵营
* 元宝数量
* 血牌数量
* 点卡数量
* 特殊状态

是否公开由具体模块决定。

### 4.4 初始摸牌

玩家从基础游戏牌中获取初始手牌。

部分公开整理将数量记录为六张，但在未完成剧集逐帧核对前，具体数量暂定为可配置参数。

初始手牌应至少覆盖三类选择：

* 一张可以主动使用的行动牌
* 一张可以响应他人的防御或干扰牌
* 一张与角色成长或资源相关的卡牌

这样可以避免玩家开局只能移动而没有决策。

### 4.5 创建地图与出生

上帝根据本局启用模块创建创界大陆。

玩家被放置在一个或多个出生区域。

出生阶段完成后，系统宣布：

**开天辟地，游戏开始。**

---

## 5. 世界阶段大循环

整局游戏分为五个世界阶段。

### 5.1 第一阶段：开天

开天阶段是规则建立期。

玩家主要进行：

* 熟悉角色
* 确认身份能力
* 获得基础手牌
* 选择初步路线
* 完成新手事件
* 接触世界牌和昼夜规则

这一阶段的特点是：

* 地图相对安全
* 裂变值增长缓慢
* 玩家之间暂时缺乏强烈冲突
* 大多数身份尚未暴露
* 创界山仍被完全封锁

开天阶段不能持续过久，一般只占整局前 `10%` 至 `15%`。

### 5.2 第二阶段：辟地

辟地阶段是地图探索和角色成长的主要时期。

玩家开始：

* 掷骰或使用移动力
* 探索不同区域
* 获取元宝和装备
* 参与局部游戏
* 完成任务
* 建立产业
* 接触阵营
* 进行交易
* 使用卡牌
* 积累创界资格

“辟地”不只是探索已有地图。

部分区域会在玩家首次进入后才正式生成，因此玩家实际上参与了世界的形成。

例如：

```text
首次进入未知区域
→ 上帝翻开区域牌
→ 决定地形
→ 加载对应游戏文明
→ 创建 NPC、事件与奖励
→ 将区域永久加入本局地图
```

### 5.3 第三阶段：争界

争界阶段是玩家互动最激烈的时期。

此时玩家已经拥有：

* 明确成长方向
* 较多卡牌
* 一定元宝
* 装备和技能
* 阵营关系
* 部分创界资格
* 可争夺的地图资源

主要行为包括：

* 玩家战斗
* 地盘争夺
* 交易和欺诈
* 身份推理
* 夜间暗杀
* 联盟
* 背叛
* 竞拍
* 局部游戏挑战
* 阻止他人完成资格

争界阶段会快速提高裂变值。

### 5.4 第四阶段：裂变

当裂变值达到阈值后，大地裂变正式开始。

玩家原来的循环：

```text
探索
→ 成长
→ 积累
```

会转变为：

```text
判断危险
→ 抢救资源
→ 重新规划路线
→ 应对规则变化
→ 争夺有限资格
→ 接近创界山
```

裂变阶段会发生：

* 地图板块损毁
* 道路断裂
* 安全区域收缩
* 商店关闭或涨价
* 卡牌效果变化
* 怪物强化
* 身份逐渐暴露
* 玩家死亡风险提高
* 创界山道路开始出现

裂变不能只是惩罚落后玩家。

它也必须破坏领先玩家已经形成的稳定优势。

### 5.5 第五阶段：创界

创界阶段是最终竞争。

玩家不再以长期经营为主要目标，而要集中完成：

* 进入创界山
* 争夺山门资格
* 通过最终试炼
* 击败守卫或魔王
* 处理个人与阵营目标冲突
* 决定世界最终归属

创界山不采用“第一个到达者自动获胜”的单一规则。

到达创界山意味着进入终局，而不是立即结束游戏。

---

## 6. 标准轮次核心循环

每一轮由上帝主持，包含七个阶段。

1. 上帝判定阶段
2. 世界状态阶段
3. 黑夜或秘密行动阶段
4. 黎明公开阶段
5. 玩家主要行动阶段
6. 世界事件阶段
7. 结束与裂变阶段

---

## 7. 第一阶段：上帝判定

每回合开始时，上帝从牌堆顶部翻开一张牌，通过花色或牌面进行判定。

这张牌称为“世界判定牌”。

它决定本轮的基础世界状态。

经典模式优先保留传统扑克牌花色判定，以体现陆展博将扑克规则直接嵌入世界循环的设计。

暂定对应关系如下：

| 花色 | 主要世界状态 | 核心体验 |
| --- | --- | --- |
| 黑桃 | 黑夜 | 隐藏身份与秘密行动 |
| 红桃 | 白昼或庆典 | 交易、治疗与社交 |
| 方块 | 财富之日 | 商业、元宝与地产 |
| 梅花 | 事件或冲突 | 战斗、任务与特殊变化 |

黑桃对应黑夜的规则在公开整理中较为一致；其余花色与效果需要结合剧中台词和画面进一步确认。

除了花色，点数也可以决定强度。

例如：

* 小点数代表普通状态
* 人头牌（J,Q,K）代表特殊角色介入
* `A` 代表规则反转
* 大小王代表上帝事件或裂变异常

---

## 8. 第二阶段：世界状态生效

上帝公布世界牌后，对本轮施加全局规则。

例如：

### 8.1 黑夜

* 大部分公共建筑关闭
* 夜间角色获得行动
* 普通玩家不能查看秘密行为
* 暗杀、偷窃和预言规则启用
* 部分怪物增强

### 8.2 白昼

* 玩家可以进行公开交易
* 城镇正常开放
* 公开战斗和任务正常进行
* 玩家可以发言、投票或交换情报

### 8.3 平安夜

游戏存在平安夜及礼物相关机制。

平安夜期间：

* 主动伤害受到限制
* 黑夜暗杀暂时关闭
* 每名玩家准备或获得礼物
* 玩家参与礼物交换
* 部分礼物隐藏真实效果
* 礼物可能带来元宝、道具、状态或恶作剧
* 社交类身份可以获得额外收益

### 8.4 财富之日

* 商店刷新
* 地产产生收益
* 拍卖行开放
* 赌博奖励提高
* 商业玩家获得额外行动

### 8.5 裂变异常

* 地图变化
* 路径封锁
* 局部规则反转
* 新的终局通道出现

---

## 9. 第三阶段：秘密行动循环

黑夜中玩家闭眼并将手牌置于桌面，由上帝按照特定顺序依次唤醒不同角色，完成身份任务。

公开整理记录的顺序包括杀手、妖侠、法师、盗贼、魔王、神仙、长老等，但不同版本对角色数量和顺序存在差异。

数字版中不需要所有玩家真正闭眼。

黑夜开始后，每个具有秘密权限的玩家进入独立界面。

秘密行动采用以下循环：

```text
上帝唤醒身份组
→ 玩家查看可执行行动
→ 选择目标或放弃
→ 秘密提交
→ 上帝暂存结果
→ 下一个身份组行动
→ 统一处理冲突
→ 黎明公布可见结果
```

### 9.1 暂定身份行动顺序

在完成原剧核对前，暂定基础顺序为：

```text
预知类
→ 守护类
→ 控制类
→ 偷窃类
→ 攻击类
→ 魔王类
→ 神仙类
→ 长老与裁定类
```

剧中名称可以映射到这些功能组：

* 法师：预知、施法或控制
* 妖侠：追踪、移动或干扰
* 盗贼：偷取手牌、元宝或道具
* 杀手：选择暗杀目标
* 魔王：施加全局邪恶效果
* 神仙：祝福、复活或改变判定
* 长老：裁定、封印或查看部分信息

具体能力在 [05-角色系统.md](./05-角色系统.md) 中确定。

### 9.2 黑夜信息原则

黎明时只公开结果，不公开全部过程。

例如：

公开：
昨夜有一名玩家受到袭击，但没有死亡。

不公开：

* 谁发动了袭击
* 谁进行了保护
* 保护使用了什么卡牌

这样才能保留身份推理和玩家讨论。

---

## 10. 第四阶段：黎明公开

秘密行动结束后，上帝宣布黎明结果。

可能包括：

* 玩家受伤或死亡
* 元宝被盗
* 手牌丢失
* 某区域出现异常
* 某名玩家获得祝福
* 魔王力量增长
* 身份线索出现
* 昨夜为平安夜，无人受伤

随后进入公开讨论窗口。

玩家可以：

* 陈述昨夜经历
* 分享或伪造情报
* 指控其他玩家
* 提议交易
* 形成临时联盟
* 发起规则允许的投票

公开讨论需要限时，防止一轮游戏无限延长。

---

## 11. 第五阶段：玩家主要行动循环

公开阶段是玩家日常决策的主体。

每名玩家的回合由以下步骤组成：

```text
回合开始
→ 状态结算
→ 移动
→ 区域进入
→ 主要行动
→ 卡牌与响应
→ 区域结算
→ 回合结束
```

### 11.1 回合开始

系统处理：

* 持续伤害
* 回合恢复
* 技能冷却
* 任务提示
* 区域效果
* 强制行动
* 身份特殊效果

### 11.2 移动

游戏使用棋盘和骰子，玩家根据上帝指引移动，并可能在途中触发特殊指令。

部分整理明确提到二十面骰，但具体骰子数量和使用范围仍需核实。

移动可以由以下方式共同决定：

* 基础移动力
* 掷骰结果
* 职业修正
* 坐骑
* 卡牌
* 区域规则
* 传送点

并非所有区域都使用相同移动方式。

例如：

* 普通区域使用移动点
* 飞行棋区域使用骰子
* 战棋区域使用格子和控制区
* 大富翁城邦沿固定道路移动
* 地下城逐房间推进
* 创界山使用独立登山路线

### 11.3 进入区域

玩家停留在某个区域后，区域可能触发：

* 强制事件
* 地形伤害
* NPC 互动
* 商店
* 战斗
* 局部游戏
* 任务
* 地产费用
* 真心话大冒险
* 隐藏入口
* 创界资格检查

### 11.4 主要行动

玩家每回合可从以下行动中选择有限数量：

* 攻击
* 使用卡牌
* 使用技能
* 购买
* 出售
* 交易
* 调查
* 接受任务
* 挑战副本
* 参与局部游戏
* 建造
* 占领
* 休息
* 发起真心话大冒险
* 与上帝或规则实体签订契约

### 11.5 响应窗口

当玩家执行可被干预的行动时，系统开启响应窗口。

其他玩家可以：

* 防御
* 取消
* 转移
* 加倍
* 改变目标
* 改变花色
* 使用装备
* 发动身份能力

响应顺序和冲突处理将在战斗与卡牌系统中详细定义。

---

## 12. 遭受攻击的原作循环

网络整理版本普遍提到：玩家遭受攻击时，可以在“弃掉全部手牌”和“回答真心话”之间进行选择。

这一规则非常符合陆展博将战斗游戏与夜店游戏硬性融合的设计方式，建议在确认剧集内容后保留。

攻击命中后，防守玩家进入“危机选择”：

* 正常承受伤害
* 弃掉全部手牌，然后减免部分伤害
* 接受真心话，公开回答系统或攻击者提出的问题，再减免部分伤害
* 接受大冒险，获得一个限时任务，成功后减免或返还伤害

为避免玩家被迫披露现实隐私，数字版中的“真心话”应使用游戏内问题，例如：

* 你的隐藏阵营是否与当前盟友一致？
* 你的手牌中是否有攻击牌？
* 你是否进入过魔王领？
* 你是否持有创界印记？
* 你上轮是否对某玩家使用过秘密行动？

回答可能为真，也可能允许说谎，但说谎被识破后产生规则惩罚。

---

## 13. 元宝循环

玩家通过以下行为获得元宝：

* 完成任务
* 赢得局部游戏
* 击败怪物
* 出售物品
* 收取地产收益
* 完成真心话或大冒险
* 接受契约
* 身份奖励
* 世界牌效果

玩家使用元宝：

* 购买装备
* 学习技能
* 参与赌博
* 发起拍卖
* 转职
* 解锁道路
* 支付地产费用
* 获取创界资格
* 触发特殊社交行动

部分整理提到，玩家拥有五个元宝后，可以消耗元宝命令其他玩家进行真心话或大冒险。

该机制可以形成一个明确循环：

```text
参与事件和小游戏
→ 获得元宝
→ 消耗元宝影响其他玩家
→ 制造信息、任务或风险
→ 改变联盟和身份判断
```

为避免元宝只成为普通货币，经典模式应保留至少一种“元宝直接购买社交权力”的功能。

---

## 14. 点卡循环

游戏中存在点卡。

点卡是一种稀有的规则例外资源。

玩家获得点卡的途径包括：

* 高难度任务
* 世界事件
* 特殊商店
* 上帝奖励
* 创界山支线
* 神鬼契约
* 放弃某种长期权利

玩家可以使用点卡：

* 重新掷骰
* 补充手牌
* 复活
* 中途加入
* 解除部分限制
* 进入特殊副本
* 跳过等待
* 修改一次角色配置
* 请求上帝重新判定

网络整理版本中出现了“中途加入需要购买点卡”“可以购买较小点卡换用六面骰”等具有明显喜剧效果的规则。

该规则是否完整进入经典模式需要进一步核实，但“点卡可以购买规则便利”应当保留。

---

## 15. 局部游戏循环

陆展博的设计核心不是只融合一种 RPG 规则，而是融合大量网络游戏、桌面游戏和夜店游戏。

因此，玩家在地图中会不断进入短时局部游戏。

### 15.1 局部游戏的统一结构

任何局部游戏都必须遵循：

```text
触发
→ 展示规则
→ 确定参与者
→ 支付成本或押注
→ 进行 1 至 3 轮核心决策
→ 判定胜负
→ 奖励与惩罚进入主世界
→ 返回主循环
```

单次局部游戏应控制在主世界一轮以内，避免其他玩家长时间等待。

### 15.2 卡牌对抗模块

吸收：

* 三国杀
* UNO
* 扑克牌
* 集换式卡牌游戏
* 牌库构筑游戏

核心结果可以是：

* 获得手牌
* 抢夺装备
* 改变行动顺序
* 获得花色标记
* 获取身份线索

### 15.3 身份推理模块

吸收：

* 狼人杀
* 天黑请闭眼
* 阵营游戏
* 剧本杀
* 国王游戏

核心结果可以是：

* 揭示身份
* 放逐玩家
* 获得阵营权力
* 取消某玩家夜间行动
* 产生错误指控惩罚

### 15.4 地产经营模块

吸收：

* 大富翁
* 模拟经营
* 股票游戏
* 拍卖游戏
* 工人放置

核心结果可以是：

* 获得地产
* 收取租金
* 建造商铺
* 控制区域服务
* 影响资源价格

### 15.5 赌博模块

吸收：

* 德州扑克
* 麻将
* 斗地主
* 骰子
* 二十一点
* 牌九类玩法

核心结果可以是：

* 赢取元宝
* 赢取点卡
* 输掉装备
* 获得赌神称号
* 欠下债务或契约

### 15.6 冒险副本模块

吸收：

* MMORPG
* 地下城探索
* 肉鸽游戏
* 跑团
* 合作 Boss 战

核心结果可以是：

* 获得装备
* 提升等级
* 获得神器
* 获得创界印记
* 提高裂变值

### 15.7 聚会游戏模块

吸收：

* 真心话大冒险
* 礼物交换
* 国王游戏
* 猜词
* 社交挑战

核心结果可以是：

* 获取情报
* 建立或破坏关系
* 获得元宝
* 施加限时任务
* 暂时控制其他玩家行动

### 15.8 战棋与区域控制模块

吸收：

* 中国象棋
* 国际象棋
* 战棋
* 区域控制
* 塔防

核心结果可以是：

* 占领道路
* 获得城池
* 控制地图通行
* 获取军事资源
* 阻止其他玩家登山

---

## 16. 第六阶段：世界事件结算

所有玩家完成主要行动后，上帝处理非玩家单位与世界变化。

包括：

* 怪物行动
* NPC 行动
* 商店刷新
* 地产收益
* 任务倒计时
* 区域控制变化
* 魔王势力推进
* 神仙干预
* 未完成大冒险惩罚
* 礼物效果
* 契约结算

该阶段让世界在玩家不主动操作时也会变化。

---

## 17. 第七阶段：结束与裂变检查

每轮最后执行：

```text
持续状态减少
→ 临时卡牌失效
→ 玩家死亡检查
→ 身份目标检查
→ 阵营目标检查
→ 创界资格检查
→ 裂变值增长
→ 世界阶段检查
→ 胜负检查
```

裂变值可以因以下行为增长：

* 玩家战斗
* 角色死亡
* 使用点卡修改规则
* 魔王行动
* 大型副本完成
* 地产垄断
* 多个规则同时冲突
* 世界牌特殊效果
* 达到固定轮数
* 玩家获得多个创界印记

这样，玩家使用越多复杂系统，世界越容易不稳定。

这符合整个游戏的主题：

**所有规则都很有趣，但规则越多，世界越容易裂变。**

---

## 18. 成长循环

玩家的长期成长由五条相互关联的路线组成。

```text
角色成长
├── 等级成长
├── 卡牌成长
├── 经济成长
├── 身份成长
└── 创界资格成长
```

### 18.1 等级成长

通过：

* 战斗
* 任务
* 副本
* 探索

获得：

* 属性
* 技能
* 职业进阶
* 装备槽位

### 18.2 卡牌成长

通过：

* 抽牌
* 购买
* 奖励
* 交换
* 抢夺
* 合成

形成个人手牌和牌库策略。

### 18.3 经济成长

通过：

* 元宝
* 地产
* 商铺
* 投资
* 赌博
* 拍卖

获得资源优势。

### 18.4 身份成长

通过：

* 完成秘密任务
* 获得身份线索
* 误导其他玩家
* 推动阵营目标
* 身份进阶

身份并非开局后永远不变。

某些玩家可能：

* 转化阵营
* 获得第二身份
* 暴露身份
* 放弃身份
* 成为长老
* 成为魔王代理人

### 18.5 创界资格成长

玩家通过以下方式获得创界资格：

* 创界印记
* 神器
* 阵营许可
* 特定称号
* 完成终局任务
* 控制关键区域
* 支付元宝
* 牺牲点卡
* 赢得重要局部游戏

不同玩家可以通过不同玩法走向创界山。

这样可以避免所有人都必须刷怪升级。

---

## 19. 裂变后的循环变化

裂变前的核心问题是：

**我怎样变得更强？**

裂变后的核心问题变为：

**我怎样带着有限的资源活着到达创界山？**

裂变发生后，以下常规循环会被改变：

| 裂变前 | 裂变后 |
| --- | --- |
| 自由探索 | 被迫迁移 |
| 长期经营 | 抢救资产 |
| 稳定交易 | 资源短缺 |
| 隐藏身份 | 身份逐渐曝光 |
| 多路线成长 | 路线不断关闭 |
| 反复刷副本 | 副本崩塌或变异 |
| 积累手牌 | 手牌腐化或强制弃置 |
| 等待最佳机会 | 必须迅速行动 |

裂变需要明显加速游戏。

进入裂变阶段后，不允许玩家继续无限经营。

---

## 20. 创界山终局循环

### 20.1 山门开启

当满足以下任一条件时，创界山开始开放：

* 裂变达到指定等级
* 有玩家收集足够创界印记
* 魔王解除封印
* 所有关键区域被探索
* 世界达到最大轮数
* 特殊世界牌被翻开

### 20.2 登山资格

玩家需要满足至少一类资格：

* 英雄资格：完成主要 Boss 任务
* 财富资格：支付大量元宝
* 身份资格：完成秘密目标
* 战略资格：控制关键区域
* 卡牌资格：集齐指定牌组
* 社交资格：获得多数玩家支持
* 点卡资格：支付稀有点卡
* 冒险资格：获得神器

### 20.3 登山过程

登山不是普通地图移动，而是一系列终局挑战：

```text
山门争夺
→ 规则试炼
→ 身份揭晓
→ 阵营选择
→ 最终 Boss 或玩家对抗
→ 创界权判定
```

### 20.4 最终选择

到达山顶的玩家可能需要选择：

* 修复原世界
* 创造新世界
* 成为新上帝
* 释放魔王
* 与他人共享创界权
* 独占创界山
* 关闭部分规则模块
* 让裂变继续

该选择与玩家身份、阵营和个人目标共同决定最终胜负。

---

## 21. 胜利循环

游戏不采用单一“先到终点者获胜”。

胜利分为三个层次：

### 21.1 世界胜利

决定创界大陆最终是否被修复、毁灭或重建。

### 21.2 阵营胜利

决定神仙、魔王、商会、冒险者或其他阵营是否完成目标。

### 21.3 个人胜利

决定玩家是否完成自己的隐藏任务和创界目标。

最终可能出现：

* 单人完全胜利
* 阵营共同胜利
* 多名玩家有限胜利
* 世界获救但玩家个人失败
* 玩家获胜但世界毁灭
* 魔王胜利
* 上帝判定无人有资格创界

这可以让背叛、合作和自我牺牲都具有意义。

---

## 22. 核心节奏控制

为了避免陆展博式复杂度导致一局无限延长，标准对局建议按照以下比例组织：

| 世界阶段 | 对局占比 | 主要体验 |
| --- | --- | --- |
| 开天 | 10% | 认识角色与规则 |
| 辟地 | 30% | 探索与成长 |
| 争界 | 25% | 玩家博弈与争夺 |
| 裂变 | 20% | 灾难、生存与路线压缩 |
| 创界 | 15% | 创界山终局 |

每轮至少应发生一项能够改变局势的事情：

* 新区域出现
* 玩家获得关键资源
* 身份线索暴露
* 地产易主
* 玩家受伤或死亡
* 世界规则改变
* 裂变值增长
* 创界资格被获得或失去

禁止出现连续多轮只有摸牌、移动和结束回合的情况。

---

## 23. 核心循环设计原则

### 23.1 剧中流程是主轴

上帝、发牌、身份、种族、血牌、元宝、世界判定、黑夜行动、骰子移动、真心话大冒险、点卡和平安夜不能被降级为无关彩蛋。

它们必须参与标准核心循环。

### 23.2 已有游戏提供局部乐趣

吸收几十种游戏时，不复制完整规则，而提取其最有辨识度的决策。

例如：

* UNO 提取花色与反转
* 狼人杀提取黑夜身份和推理
* 三国杀提取出牌与响应
* 大富翁提取地产和租金
* 德州扑克提取下注、欺骗和牌型
* RPG 提取职业、装备与成长
* 飞行棋提取骰子、竞速和跳跃
* 真心话大冒险提取社交风险
* 地下城游戏提取合作探索和 Boss
* 战棋提取位置、地形和控制

### 23.3 局部游戏必须返回主循环

任何小游戏结束后，都必须改变玩家下一步决策。

### 23.4 复杂规则由系统承担

玩家只需要选择：

* 做什么
* 对谁做
* 愿意付出什么
* 愿意承担什么风险

系统负责计算和解释。

### 23.5 裂变负责结束游戏

无论玩家多么喜欢经营、赌博、刷副本或推理，裂变都会逐渐关闭这些循环，并把所有人推向创界山。

### 23.6 每条成长路线都能接近终局

战斗不是唯一正确玩法。

玩家可以通过：

* 战斗
* 财富
* 卡牌
* 身份
* 社交
* 探索
* 赌博
* 区域控制

获得创界山资格。

---

## 24. 最小可玩版本的核心循环

首个可玩版本暂时不实现全部几十种游戏模块。

基础版本采用以下组合：

* RPG 角色成长
* 三国杀式卡牌与响应
* 狼人杀式黑夜身份
* 大富翁式地图与地产
* 骰子移动
* 真心话大冒险
* 地下城探索
* 世界裂变
* 创界山终局

基础对局流程为：

```text
上帝创建世界
→ 发放身份、种族、血牌、元宝和手牌
→ 翻开世界牌
→ 处理黑夜或特殊世界状态
→ 玩家掷骰或使用移动力
→ 进入区域并触发玩法
→ 使用卡牌、交易、战斗或参与小游戏
→ 获得元宝、装备和创界印记
→ 结算世界事件
→ 提高裂变值
→ 地图逐步崩塌
→ 创界山开放
→ 玩家完成终局试炼
→ 上帝判定世界、阵营和个人胜负
```

只要该循环能够稳定产生：

* 探索
* 成长
* 隐藏信息
* 玩家互动
* 荒诞事件
* 策略转折
* 裂变压力
* 创界山决战

就证明项目的核心成立。

---

## 25. 待核实与待设计内容

### 25.1 需要根据剧集继续核实

1. 初始手牌的准确数量。
2. 身份牌、种族牌、血牌和元宝的准确发放方式。
3. 世界牌四种花色的完整对应关系。
4. 黑夜中角色的准确唤醒顺序。
5. 杀手、妖侠、法师、盗贼、魔王、神仙和长老的原始能力。
6. 二十面骰是否为基础移动骰。
7. 五个元宝发动真心话大冒险的准确规则。
8. 遭受攻击时弃牌或回答真心话的准确表述。
9. 平安夜的完整礼物规则。
10. 神鬼契约、隐藏攻击和十二种攻击方式是否均属于剧中正式规则。
11. 点卡、中途加入和骰子替换的准确关系。
12. 剧中最终获胜是否只要求到达创界山，还是还存在其他判定。

### 25.2 后续文档需要解决

1. 世界牌的详细牌库。
2. 一轮中每名玩家的行动数量。
3. 地图移动和骰子的关系。
4. 血牌代表生命值还是独立资源。
5. 元宝基础产出和消费范围。
6. 点卡是否允许在经典模式中自由交易。
7. 身份、职业、种族和称号的具体区分。
8. 黑夜行动冲突的优先级。
9. 玩家死亡后的处理方式。
10. 创界山最终试炼的具体形式。

---

## 26. 核心循环总结

《开天辟地之大地的裂变》的核心循环不是简单的：

```text
掷骰
→ 移动
→ 抽牌
→ 到达终点
```

而是：

```text
上帝改变世界
→ 身份在黑夜中秘密行动
→ 玩家在白昼探索地图
→ 进入不同规则区域
→ 参与由已有游戏演化而来的局部玩法
→ 获取元宝、卡牌、装备、情报和创界资格
→ 与其他玩家合作、欺骗、交易和战斗
→ 复杂规则不断制造冲突
→ 冲突积累为大地裂变
→ 裂变摧毁稳定世界
→ 所有路线最终汇聚创界山
→ 玩家争夺新世界的规则制定权
```

它的核心乐趣并不来自某一种游戏机制，而来自多种规则在同一个世界中发生碰撞。

玩家每一轮都在回答三个问题：

1. 当前世界允许我做什么？
2. 我应该利用哪一种游戏规则获得优势？
3. 在大地彻底裂变之前，我怎样抵达创界山？

这就是《开天辟地之大地的裂变》的核心循环。
