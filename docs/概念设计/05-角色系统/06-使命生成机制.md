# 《开天辟地之大地的裂变》使命生成机制

## 1. 概述

使命（Mission）是角色在一局游戏中需要完成的个人目标集合，也是角色信仰、身份以及随机事件系统在实际玩法中的综合体现。

为了提高可玩性、隐藏性与重复游玩价值，本作不采用“单一固定使命”的设计，而是将角色使命拆分为多个目标层共同组成。

每名角色的使命由以下三类目标共同构成：

其中：

* 公共目标：所有玩家共享的主线目标；
* 角色目标：基于隐藏信仰与公开身份生成的个人目标；
* 随机目标：从任务池中随机抽取的额外个人目标。

因此，角色的完整使命可以理解为：

`使命 = 公共目标 + 角色目标 + 随机目标`

其中，公共目标具有公开性，角色目标与随机目标主要对角色本人可见。其他玩家无法直接查看角色的完整使命内容，只能根据其公开身份和游戏行为进行推测。

---

## 2. 信息公开规则

角色相关信息分为公开信息与隐藏信息。

| 信息 | 是否公开 |
|------|------|
| 基础身份 | 公开 |
| 转职结果 | 公开 |
| 种族 | 公开 |
| 性别 | 公开 |
| 公共目标 | 公开 |
| 信仰 | 隐藏 |
| 角色目标 | 隐藏 |
| 随机目标 | 隐藏 |
| 使命完成进度 | 原则上隐藏 |

玩家可以知道其他角色是法师、游侠或魔王，但无法确定其属于光明、混沌还是平衡信仰。

因此，即使两名玩家拥有相同身份，也可能因为信仰不同与随机目标不同，而承担完全不同的个人使命。

---

## 3. 生成逻辑

游戏开始后，系统依次完成以下流程：

```text
生成公共目标
        │
        ▼
确定隐藏信仰
        │
        ▼
读取转职前基础身份
        │
        ▼
生成角色目标
        │
        ▼
从随机目标池中抽取一个或多个随机目标
        │
        ▼
组合为完整使命
        │
        ▼
向玩家展示公共目标与个人目标信息
```

### 3.1 公共目标

公共目标是所有玩家共享的主线任务。

由于本作尊重《爱情公寓》中《开天辟地之大地的裂变》的原始设定，因此公共目标通常围绕创界山展开，例如：

* 攀登创界山；
* 推进创界山相关阶段任务；
* 击败创界山守关首领；
* 完成终局创界仪式。

公共目标决定的是整局游戏为什么会不断向终局推进。

### 3.2 角色目标

系统应为每一种“信仰与身份组合”准备若干可选角色目标，并从中随机抽取一个。

例如：

光明信仰 + 游侠身份  
→ 生成偏向探索、救援或守护区域的角色目标

混沌信仰 + 游侠身份  
→ 生成偏向争夺资源、破坏秩序或挑战强者的角色目标

平衡信仰 + 游侠身份  
→ 生成偏向维持地图状态、限制阵营扩张或保护自然的角色目标

### 3.3 随机目标

随机目标来自独立任务池。

游戏开始时，每名玩家随机获取一个或多个任务，并将其组合为自己的随机目标。

这些随机目标不直接绑定某个固定身份或信仰，而是用于增加：

* 对局随机性；
* 玩家行为差异；
* 目标推理难度；
* 重复游玩价值。

以上仅用于说明生成方向，具体目标内容将在后续任务与事件设计中补充。

---

## 4. 信仰与身份的职责

### 4.1 信仰决定“为什么行动”

不同信仰决定使命所服务的最终目标。

* 光明使命通常偏向守护、救援与恢复秩序；
* 混沌使命通常偏向变革、争夺与打破现状；
* 平衡使命通常偏向限制极端、维持循环与控制局势。

### 4.2 身份决定“如何行动”

基础身份决定角色更适合通过什么方式完成使命。

例如：

* 法师更可能通过法术、控制或特殊仪式完成使命；
* 杀手更可能通过追踪、击败特定目标或隐秘行动完成使命；
* 盗贼更可能通过资源获取、潜入或交易完成使命；
* 游侠更可能通过探索、移动或地图互动完成使命；
* 魔王更可能通过正面压制、占领或挑战完成使命；
* 神仙长老更可能通过支援、调解或影响其他角色完成使命。

---

## 5. 与转职的关系

角色目标生成时读取的是角色的转职前基础身份。

角色在游戏中后期完成转职后，原有角色目标与随机目标不会被替换，也不会重新生成。

转职可以为角色提供新的能力和完成手段，但不会改变使命的核心目标结构。

例如，一名游侠在转职后仍然执行由“游侠身份与隐藏信仰”生成的使命，只是可以利用新的转职能力更有效地完成任务。

这样可以避免玩家为了使命收益而临时选择转职路线，也能保证一局游戏中的个人目标保持稳定。

---

## 6. 保密与推理

除公共目标外，个人使命默认只有角色本人能够查看。

玩家不得直接向其他玩家展示自己的使命卡或使命界面，但可以通过谈判、谎言、暗示或行动诱导其他玩家。

其他玩家可以依据以下信息进行推测：

* 当前公共目标；
* 角色公开身份；
* 角色的移动与战斗行为；
* 对事件与阵营冲突的选择；
* 对其他玩家的帮助或阻碍；
* 对特定地图区域及资源的关注。

因此，使命系统不仅提供个人目标，也会形成隐藏身份、行为推理与玩家博弈。由于随机目标的存在，其他玩家无法仅通过枚举“信仰 + 身份”的方式暴力推断某名角色的完整目标。

---

## 7. 完成与结算

使命可以作为角色胜利条件的一部分，也可以在完成后提供额外收益。

具体结算形式可包括：

* 获得使命积分；
* 获得经验或特殊奖励；
* 推进个人胜利条件；
* 解锁特殊事件或结局；
* 在最终结算时提供额外评分。

不同类型目标可以采用不同结算方式：

* 公共目标主要推动主线进度与终局开启；
* 角色目标主要影响个人胜利条件与信仰表达；
* 随机目标主要提供额外奖励、评分或局部优势。

使命完成后是否立即公开，可根据具体目标决定。

普通个人目标可以继续保持隐藏，直到游戏结束统一揭示；部分具有重大世界影响的目标，可以在完成时公开并触发相应事件。

---

## 8. 设计原则

使命生成机制遵循以下原则：

* 公共目标公开，个人目标隐藏；
* 信仰隐藏，身份公开，行为可被观察；
* 同一身份因信仰不同而拥有不同角色目标；
* 同一信仰因身份不同而拥有不同完成方式；
* 随机目标用于增加不可枚举性与重复游玩价值；
* 转职改变能力，不改变原始个人目标；
* 使命应引导玩家行动，但不应强迫玩家采用唯一策略；
* 不同目标的难度与最终收益应保持整体平衡。

使命系统的核心体验可以概括为：

其他玩家知道你能够做什么，也知道世界此刻要走向哪里，却不知道你真正为什么这样做。
