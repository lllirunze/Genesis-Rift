# 《开天辟地之大地的裂变》角色系统总览

## 1. 概述

角色系统是《开天辟地之大地的裂变》中用于描述玩家角色身份信息的核心系统，它决定了角色在游戏中的身份定位、成长方向以及剧情互动方式。

本章节主要描述角色本身的组成结构，不涉及具体数值、战斗公式、装备效果等内容。所有属性计算均在《11-数值系统》中进行说明。

角色系统遵循**单一职责**原则，将身份、种族、阵营等不同维度进行解耦，使各系统能够独立设计、独立扩展，同时保证整体逻辑的一致性。

---

## 2. 角色组成

一个完整的角色由以下几个部分组成：

| 模块 | 说明 |
|------|------|
| 阵营（信仰） | 决定角色的世界立场、价值观以及部分使命方向。 |
| 身份 | 角色的核心职业定位，继承《爱情公寓》原剧设定，是角色玩法的核心。 |
| 种族 | 决定角色的先天特征与成长倾向。 |
| 性别 | 角色的基础标签，仅影响部分剧情、事件及交互，不影响数值平衡。 |
| 转职 | 游戏过程中角色的发展方向，每个身份拥有多个二转分支。 |
| 使命 | 根据世界主线、阵营及身份自动生成的长期目标。 |

---

## 3. 设计原则

角色系统遵循以下设计原则：

- **尊重原剧设定**  
  身份系统以《爱情公寓》中陆展博设计的《开天辟地之大地的裂变》为基础，尽可能保留原有职业与世界观设定，并在此基础上进行合理扩展。

- **职责单一**  
  本章节仅负责描述角色身份结构，不负责任何属性计算、战斗规则及数值平衡。

- **模块解耦**  
  阵营、身份、种族、转职等模块彼此独立，可以单独调整与扩展，而不会影响其他模块。

- **易于扩展**  
  后续版本可在不破坏现有结构的前提下增加新的身份、种族、信仰裂变或命格等玩法。

---

## 4. 角色系统结构

```text
角色
├── 阵营（信仰）
├── 身份
├── 种族
├── 性别
├── 转职
└── 使命
```

上述各模块将在本章节后续文档中分别进行介绍。

> **说明：**  
> 角色的基础属性、派生属性及各种数值计算规则，不属于角色系统的职责范围，将统一放置于《11-数值系统》中进行说明。
