# 《开天辟地之大地的裂变》回合结构

## 1. 文档目的

本文档定义《开天辟地之大地的裂变》一轮游戏的完整执行顺序，包括：

* 上帝如何推动游戏
* 世界牌如何决定本轮环境
* 黑夜与白昼如何切换
* 不同身份如何依次行动
* 玩家如何移动、出牌、战斗和交易
* 局部游戏如何插入主回合
* 响应、打断和规则冲突如何处理
* 回合结束后如何推进裂变和终局
* 联机状态下等待、超时和掉线如何处理

回合结构的首要目标不是追求极简，而是：

保留陆展博原本复杂、夸张、融合多种游戏的设计，同时让每一步都有稳定时序，并能够由系统自动执行。

---

## 2. 原剧回合规则基础

### 2.1 上帝负责推动流程

游戏中由玩家推选一人担任“上帝”。

上帝负责：

* 宣布游戏开始
* 发放身份牌
* 发放种族牌
* 发放血牌
* 发放元宝
* 管理基础游戏牌
* 翻开牌堆顶部卡牌
* 根据花色进行世界判定
* 在黑夜中依次唤醒不同身份
* 指引玩家完成身份任务

公开规则整理普遍记载，玩家会获得身份牌、种族牌、血牌与元宝，并从基础牌堆摸取六张初始手牌。每回合开始时，上帝翻开牌堆顶部卡牌查看花色；若为黑桃，本轮进入黑夜。

### 2.2 黑夜秘密行动

黑夜开始后：

* 所有玩家闭眼
* 玩家将手牌置于桌面
* 上帝依次唤醒特定身份
* 被唤醒的身份完成自己的任务
* 其他玩家不能查看其行动

公开整理中出现的顺序为：

```text
杀手
→ 妖侠
→ 法师
→ 盗贼
→ 魔王
→ 神仙
→ 长老
```

部分整理将“神仙长老”视为一个整体身份，另一些则将神仙与长老分开，因此具体角色边界仍需在角色系统设计时核实。

### 2.3 地图、骰子与特殊格子

游戏包含棋盘、骰子和地图移动。

网络规则整理中普遍出现二十面骰、行走途中触发房租、真心话大冒险等特殊指令，但由于这些资料并非正式规则书，本文将“使用骰子移动和触发格子”视为原剧核心方向，将具体骰面与格子效果暂列为待核实。

### 2.4 元宝、真心话大冒险与点卡

游戏中存在：

* 元宝
* 真心话
* 大冒险
* 点卡
* 特殊契约
* 中途加入等非常规规则

网络整理普遍提到，玩家遭受攻击时可在弃掉全部手牌和回答真心话之间选择；获得一定元宝后，可以命令其他玩家进行真心话或大冒险。该规则非常符合剧中将桌游、网游和夜店游戏混合的整体设定，但其精确数值仍需后续核实。

## 3. 回合结构设计原则

### 3.1 原剧时序优先

原剧明确出现的以下流程不能被改变：

```text
上帝判定
→ 根据花色改变世界状态
→ 黑夜身份秘密行动
→ 玩家按照棋盘规则继续游戏
```

合理补全只能发生在原剧没有详细说明的部分。

### 3.2 一轮不等于一名玩家的回合

本游戏区分：

世界轮

所有玩家共同经历一次世界牌判定、秘密阶段、公开行动和世界结算。

玩家回合

某名玩家在公开行动阶段中的个人行动时间。

因此：

```text
1 个世界轮
=
1 次上帝判定
+ 1 次秘密阶段
+ 所有玩家各自的公开回合
+ 1 次世界结算
```

### 3.3 世界状态先于个人行动

玩家不能先行动，再判断今天是白天还是黑夜。

每轮开始必须由上帝先改变世界规则。

### 3.4 秘密行动先于公开行动

黑夜身份的行动通常在公开行动前结算。

这样玩家在白昼阶段需要根据昨夜发生的结果重新判断局势。

### 3.5 插入小游戏不能破坏主时序

局部游戏可以插入玩家回合，但必须：

* 有明确触发点
* 有参与人数限制
* 有时间限制
* 有返回主回合的位置
* 有主世界奖励或惩罚

### 3.6 服务端上帝负责最终时序

联机模式中，客户端不能自行决定：

* 当前处于哪个阶段
* 谁可以行动
* 响应窗口是否关闭
* 随机结果
* 黑夜行动顺序
* 最终结算结果

所有阶段转换均由系统上帝统一推进。

## 4. 回合层级

完整时序分为四层。

```text
对局
└── 世界阶段
    └── 世界轮
        └── 玩家回合与响应窗口
```

### 4.1 对局

一局游戏从玩家进入房间开始，到创界山结算结束。

### 4.2 世界阶段

一局游戏依次经历：

```text
开天阶段
→ 辟地阶段
→ 争界阶段
→ 裂变阶段
→ 创界阶段
```

世界阶段会修改回合规则。

例如：

* 开天阶段黑夜行动较少
* 争界阶段允许更多玩家对抗
* 裂变阶段每轮增加灾难结算
* 创界阶段取消普通经营行动

### 4.3 世界轮

世界轮是标准游戏节奏单位。

建议标准对局由约 `12` 至 `20` 个世界轮构成，具体数量由地图规模、玩家人数和裂变速度决定。

### 4.4 玩家回合

每名玩家在公开行动阶段获得一次正常玩家回合。

特殊效果可能带来：

* 额外回合
* 跳过回合
* 延迟回合
* 与其他玩家交换顺序
* 同时行动
* 插入行动

---

## 5. 标准世界轮总览

一个标准世界轮由九个阶段组成。

0. 轮次准备阶段
1. 上帝判定阶段
2. 世界状态公布阶段
3. 黑夜与秘密行动阶段
4. 黎明结算阶段
5. 公开讨论阶段
6. 玩家行动阶段
7. 区域与世界行动阶段
8. 裂变与终局检查阶段
9. 轮次结束阶段

不是每一轮都会完整执行全部阶段。

例如：

* 没有黑夜时可以跳过秘密行动阶段
* 平安夜可能用礼物阶段替换攻击类秘密行动
* 裂变阶段会额外加入灾难步骤
* 创界阶段会用登山行动替换普通地图行动

---

## 6. 第 0 阶段：轮次准备

轮次准备阶段用于处理上一轮遗留状态。

系统依次执行：

```text
确认当前轮数
→ 确认世界阶段
→ 检查在线玩家
→ 处理延迟效果
→ 恢复每轮资源
→ 减少持续时间
→ 重置行动次数
→ 检查强制事件
```

### 6.1 行动资源恢复

每名玩家重置：

* 主要行动
* 次要行动
* 移动力
* 响应次数
* 特殊职业资源

### 6.2 持续状态更新

例如：

* 中毒剩余 2 轮变为 1 轮
* 隐身在本轮开始时失效
* 大冒险任务剩余时间减少
* 贷款或契约产生利息
* 冷却完成的技能重新可用

### 6.3 掉线确认

若玩家掉线：

* 短时掉线保留席位
* 系统等待有限时间
* 超时后按照该玩家预设策略托管
* 涉及隐藏信息时，托管系统不得泄露其身份

---

## 7. 第 1 阶段：上帝判定

每轮开始时，上帝从基础牌堆或世界牌堆顶部翻开一张牌，根据其花色进行判定。

### 7.1 判定牌组成

判定牌至少具有：

* 花色
* 点数
* 特殊牌型
* 是否为人头牌
* 是否为 `A`
* 是否为大小王

花色决定基础环境。

点数决定环境强度、持续时间或附加效果。

### 7.2 花色规则

原剧目前可以较高把握确认的是：

```text
黑桃
→ 黑夜
```

其余花色的完整含义尚无可靠正式规则，因此暂按以下结构进行合理补全。

黑桃：黑夜

* 开启秘密身份行动
* 夜间职业生效
* 部分公共区域关闭
* 暗杀和偷窃可能发生
* 夜行怪物增强

红桃：白昼或庆典

* 恢复与社交效果增强
* 城镇功能正常开放
* 礼物、交易与结盟效果增强
* 特殊点数可能进入平安夜

方块：财富与资源

* 商店刷新
* 地产产生收益
* 元宝事件增加
* 赌博与拍卖模块活跃
* 资源价格可能变化

梅花：冲突与冒险

* 怪物活跃
* 副本事件增强
* 战斗奖励提高
* 地图事件数量增加
* 特殊点数可能触发强制挑战

以上红桃、方块和梅花含义属于合理补全，并非直接声称是原剧完整规则。

### 7.3 点数规则

点数用于进一步修饰花色。

`2` 至 `10`

表示普通强度。

点数越高，世界效果越强。

例如：

* 黑桃 `3`：普通黑夜
* 黑桃 `10`：长夜，夜间身份获得强化

`J`、`Q`、`K`：人头牌

人头牌代表特殊世界人物介入。

* `J`：执行者、侍从或刺客
* `Q`：女王、神女或命运角色
* `K`：国王、魔王或统治者

具体效果由花色共同决定。

例如：

* 黑桃 `K`：魔王之夜
* 红桃 `Q`：女神庆典
* 方块 `K`：黄金王令
* 梅花 `J`：挑战者降临

`A`：反转牌

`A` 代表世界规则发生反转或升级。

例如：

* 白昼变为黑夜
* 治疗转化为伤害
* 地产收益改为维护费用
* 顺时针行动改为逆时针
* 胜者与败者奖励交换
* 角色能力发生临时反转

`A` 的反转效果必须在世界状态公布阶段明确展示，不能让玩家在不知情时受罚。

大小王：上帝事件

大小王不属于普通昼夜。

它们触发上帝级事件。

小王

局部世界异常，例如：

* 某一区域规则改变
* 随机玩家交换位置
* 短暂出现特殊副本
* 所有人获得一次额外抽牌

大王

全局重大事件，例如：

* 裂变值大幅提高
* 魔王苏醒
* 创界山提前开启
* 所有身份发生一次检查
* 地图进行大规模重构

---

## 8. 第 2 阶段：世界状态公布

上帝翻牌后，必须向所有玩家公布当前轮次规则。

界面至少显示：

```text
当前轮次：第 7 轮
世界牌：黑桃 8
世界状态：长夜
持续时间：本轮
主要影响：
- 启用夜间身份行动
- 夜行角色移动力 +1
- 城镇商店关闭
- 暗杀伤害 +1
```

### 8.1 公开规则原则

玩家必须在行动前知道：

* 本轮处于什么环境
* 哪些行动被允许
* 哪些区域受到影响
* 是否存在额外费用
* 是否出现规则反转
* 该状态持续多久

### 8.2 隐藏附加效果

部分世界牌可以拥有不公开的附加效果。

例如：

* 本轮魔王获得额外行动
* 某区域藏有伏击
* 平安夜礼物中存在诅咒物品

隐藏效果只向拥有查看权限的角色展示。

---

## 9. 第 3 阶段：黑夜与秘密行动

当世界状态包含“黑夜”时，进入秘密行动阶段。

### 9.1 数字版闭眼机制

数字版不要求玩家物理闭眼。

进入黑夜后：

* 公共棋盘进入夜间遮罩
* 普通玩家只能看到等待界面
* 被唤醒的玩家进入私有操作界面
* 每个身份只看到自己有权知道的信息
* 所有行动由上帝秘密保存
* 结果在黎明统一公布

界面仍可显示原剧风格提示：

**天黑请闭眼。**

### 9.2 黑夜行动总顺序

原剧整理中常见顺序为：

```text
杀手
→ 妖侠
→ 法师
→ 盗贼
→ 魔王
→ 神仙
→ 长老
```

为了保证规则冲突可以正确处理，合理补全时需要区分：

* 玩家选择顺序
* 效果实际结算顺序

选择顺序

尽量保留原剧身份唤醒顺序。

结算顺序

根据效果性质进行统一结算。

建议为：

```text
预知
→ 标记
→ 守护
→ 移动
→ 偷窃
→ 控制
→ 攻击
→ 复活与救援
→ 世界级干预
→ 长老裁定
```

这样可以避免“杀手先结算导致后续守护无意义”的问题。

原剧中的唤醒顺序仍然保留，但杀手提交的攻击先暂存，待守护、神仙和长老等相关效果完成后统一结算。

## 10. 黑夜身份行动细则

### 10.1 杀手

杀手在黑夜中被上帝唤醒并完成身份任务。

杀手可以：

* 选择一名同区域或攻击范围内的玩家
* 发动暗杀
* 放弃行动
* 使用隐藏攻击
* 配合某些卡牌伪造袭击来源

杀手选择目标时不会立即造成死亡，攻击结果将在黑夜结算时确定。

### 10.2 妖侠

妖侠属于黑夜唤醒身份之一。

妖侠定位为移动、追踪和干扰型身份。

可选择以下一种行动：

* 秘密移动到相邻区域
* 追踪某名玩家
* 标记一个目标
* 干扰一次暗杀
* 查看某一区域是否有人秘密行动

妖侠不能直接知道目标的完整身份。

### 10.3 法师

法师属于黑夜行动角色。

法师可从法术中选择：

* 预言：查看一条模糊身份信息
* 封印：令目标某项能力暂时失效
* 护盾：为玩家提供魔法防护
* 幻象：制造虚假行动结果
* 转移：改变某个效果的目标

具体能力取决于职业和技能。

### 10.4 盗贼

盗贼出现在公开整理的黑夜行动顺序中。

盗贼可以尝试偷取：

* 一张随机手牌
* 一枚元宝
* 一件未装备道具
* 一条任务线索
* 一份地产收益

若目标有守卫、陷阱或反偷窃效果，盗贼可能失败并留下线索。

### 10.5 魔王

魔王是黑夜流程中的重要概念。

魔王不一定是普通职业，也可能是隐藏身份、世界 Boss 或阵营核心。

若由玩家控制，魔王可以：

* 增加裂变值
* 诅咒区域
* 强化怪物
* 诱惑玩家签订契约
* 使某张世界牌效果恶化
* 暂时封锁创界山资格

魔王的行动一般会留下世界级征兆，但不一定直接暴露操作者。

### 10.6 神仙

神仙参与黑夜流程。

神仙可执行：

* 赐福
* 保护
* 治疗
* 改变一次随机结果
* 延缓裂变
* 复活刚刚死亡的玩家
* 清除魔王诅咒

神仙不能每轮无限救援，需要消耗法力、元宝、信仰或特定卡牌。

### 10.7 长老

长老出现在黑夜行动相关概念中。

长老属于最后裁定类角色。

长老可以：

* 查看本轮是否发生规则冲突
* 封印一个过强效果
* 检查一名玩家是否违反契约
* 延迟公开一个结果
* 保护世界规则稳定
* 对神仙和魔王冲突进行最终裁定

长老一般不能直接杀死玩家。

---

## 11. 黑夜提交与超时

每个秘密身份拥有独立行动时间。

建议：

* 普通秘密行动：`30` 秒
* 复杂目标选择：`45` 秒
* 多人协同行动：`60` 秒

### 11.1 超时默认行为

超时后：

* 杀手默认放弃暗杀
* 法师默认不施法
* 盗贼默认不偷窃
* 守护类角色优先保护预设目标
* 魔王默认增加少量裂变值
* 神仙默认保留资源
* 长老默认不进行干预

系统不能随机替玩家选择高风险目标。

### 11.2 同身份多人行动

若存在多名同类身份：

* 可分别行动
* 可由阵营秘密讨论
* 可共同选择一个目标
* 可按行动优先级依次处理

房间配置必须说明同身份是否互相知道。

---

## 12. 黑夜统一结算

秘密选择完成后，上帝按以下顺序结算。

1. 检查非法目标
2. 处理目标移动
3. 处理守护和免疫
4. 处理偷窃与资源转移
5. 处理控制与封印
6. 处理攻击和暗杀
7. 处理治疗与复活
8. 处理魔王和神仙冲突
9. 处理长老裁定
10. 生成黎明公开信息

### 12.1 例子

假设：

* 杀手攻击玩家 B
* 妖侠追踪玩家 B
* 法师给玩家 B 施加护盾
* 盗贼偷取玩家 B 的元宝
* 魔王令该区域暗杀伤害 +1
* 神仙赐福玩家 B
* 长老封印魔王加成

最终可能结算为：

* 盗贼成功偷取 `1` 枚元宝
* 杀手攻击玩家 B
* 法师护盾抵挡 `2` 点伤害
* 神仙赐福抵挡 `1` 点伤害
* 魔王加成被长老封印
* 玩家 B 受到剩余伤害，但没有死亡

黎明只会向所有人公开必要结果。

---

## 13. 第 4 阶段：黎明结算

黑夜结束后，上帝宣布：

**天亮了，请睁眼。**

### 13.1 公开结果

公开信息可以包括：

* 哪些玩家受伤
* 是否有人死亡
* 是否有元宝失窃
* 哪些区域遭到诅咒
* 是否发生平安夜
* 是否有怪物出现
* 是否有身份被公开
* 是否发生世界异常

### 13.2 不公开的信息

默认不公开：

* 谁发动了攻击
* 谁提供了保护
* 谁偷取了资源
* 谁改变了判定
* 某个效果由什么具体身份造成

### 13.3 死亡处理

死亡玩家不应立即永久退出漫长对局。

角色死亡后进入“濒死结算”。

可能处理方式包括：

* 消耗血牌
* 使用治疗牌
* 神仙复活
* 支付点卡复活
* 接受神鬼契约
* 变为灵魂状态
* 由队友救援
* 转化为亡灵种族

若所有救援均失败，角色才进入正式死亡状态。

正式死亡后，玩家仍可：

* 以灵魂形式提供有限行动
* 接管中立角色
* 成为上帝助手
* 等待特殊复活事件

---

## 14. 第 5 阶段：公开讨论

黎明后设置有限时间的公开讨论阶段。

玩家可以：

* 讨论昨夜结果
* 指认嫌疑人
* 解释自身行为
* 提议合作
* 提供或伪造情报
* 发起交易意向
* 准备投票
* 发起身份挑战

公开讨论不能直接修改游戏状态。

正式交易、攻击和技能必须在后续行动阶段提交。

### 14.1 禁止问“为什么”的原作候选规则

部分网络整理提到：

游戏过程中不可以说“为什么”，否则罚酒一杯。

这一规则具有非常强的喜剧效果，但数字版不适合要求现实饮酒。

经典娱乐模式可以保留为“禁词规则”：

* 玩家在内置文字聊天中发送“为什么”
* 系统识别后施加一个娱乐性惩罚
* 例如失去 `1` 点行动时间、抽取恶作剧牌或完成一个小游戏
* 语音聊天不进行强制识别，以避免误判和隐私问题

该规则可以作为经典剧集彩蛋模块，不进入竞技规则。

---

## 15. 第 6 阶段：玩家行动

所有存活且未被跳过的玩家依次获得公开回合。

### 15.1 行动顺序

默认按照：

```text
起始玩家
→ 顺时针方向
```

行动。

起始玩家可以由：

* 上轮顺序延续
* 世界牌决定
* 骰子决定
* 当前速度属性决定
* 特殊卡牌改变

### 15.2 反转顺序

当世界牌、UNO 式反转牌或特殊效果触发时：

```text
顺时针
↔
逆时针
```

`A` 牌可以作为世界级顺序反转来源。

### 15.3 同时行动

部分环节适合同时进行：

* 选择礼物
* 秘密下注
* 选择探索路线
* 提交投票
* 选择阵营
* 世界灾难避难

同时行动由上帝在所有玩家提交后统一结算。

---

## 16. 玩家回合结构

每名玩家的公开回合包含六个步骤。

1. 回合开始
2. 摸牌或补给
3. 移动
4. 区域触发
5. 行动
6. 回合结束

---

## 17. 玩家回合开始

系统处理该玩家身上的：

* 中毒
* 燃烧
* 治疗
* 地产收益
* 任务倒计时
* 契约费用
* 职业技能
* 区域状态
* 强制大冒险

若玩家处于：

* 昏迷
* 监禁
* 石化
* 强制跳过
* 灵魂状态

则可能减少或失去本回合行动。

---

## 18. 摸牌或补给

玩家回合开始时默认摸取一张基础游戏牌。

部分角色或区域可以修改摸牌规则。

例如：

* 商人在黄金城多摸一张经济牌
* 法师在神仙境改摸法术牌
* 盗贼可以选择从弃牌堆顶偷取
* 手牌达到上限时不能继续摸牌
* 裂变阶段可能减少摸牌量

玩家也可以选择“补给”代替普通摸牌，例如：

* 恢复生命
* 获得基础物资
* 修理装备
* 领取地产收益

---

## 19. 移动阶段

### 19.1 默认移动方式

为了尊重原剧的骰子棋盘体验，经典模式中保留骰子移动。

标准流程：

```text
玩家声明移动
→ 上帝允许掷骰
→ 系统生成骰点
→ 玩家在可达路径中选择
→ 棋子移动
→ 途中检查特殊格
→ 到达目标区域
```

### 19.2 二十面骰

公开整理多次提到二十面骰，因此经典模式优先考虑将 `D20` 作为世界移动与高风险检定用骰。

为了避免一次掷出 `20` 导致跨越半张地图，`D20` 不直接等于移动格数。

可采用分段映射：

| D20 点数 | 基础移动 |
| --- | --- |
| 1—4 | 1 格 |
| 5—8 | 2 格 |
| 9—12 | 3 格 |
| 13—16 | 4 格 |
| 17—19 | 5 格 |
| 20 | 6 格并触发特殊效果 |

这样既保留 `D20`，也能控制地图节奏。

### 19.3 骰子替换

点卡、装备和区域规则可以允许使用：

* 六面骰
* 八面骰
* 十二面骰
* 二十面骰
* 双骰
* 固定移动值

不同骰子不一定是越大越好。

例如，某些任务要求准确掷出特定点数，小骰子反而更稳定。

### 19.4 途中触发

玩家移动过程中可能触发：

* 经过地产支付租金
* 路过商店
* 进入陷阱
* 被敌人拦截
* 触发真心话大冒险
* 获得元宝
* 进入传送格
* 抽取区域事件
* 遭遇其他玩家

---

## 20. 区域触发阶段

玩家到达区域后，按以下优先级处理：

```text
禁止进入检查
→ 经过效果
→ 到达效果
→ 地产费用
→ 强制事件
→ 怪物遭遇
→ NPC 互动
→ 自愿区域功能
```

### 20.1 强制触发

例如：

* 支付房租
* 抽事件牌
* 遭遇怪物
* 接受真心话
* 进入迷宫
* 被传送
* 受到环境伤害

### 20.2 自愿触发

例如：

* 进入商店
* 参与赌博
* 接取任务
* 发起交易
* 进入副本
* 挑战竞技场
* 使用传送阵
* 投资地产

---

## 21. 行动阶段

每名玩家默认拥有：

* `1` 次主要行动
* `1` 次次要行动
* 若干自由行动

具体次数可由角色、职业和模块改变。

### 21.1 主要行动

包括：

* 攻击
* 使用强力技能
* 参与局部游戏
* 进入副本
* 建造
* 占领
* 发起真心话大冒险
* 进行大额交易
* 尝试转职
* 发起身份挑战

### 21.2 次要行动

包括：

* 使用普通道具
* 装备或卸下物品
* 小额交易
* 调查
* 与 NPC 交谈
* 拾取物品
* 修理装备
* 转换资源

### 21.3 自由行动

包括：

* 查看公开信息
* 查看自己的卡牌
* 整理背包
* 发送聊天
* 接受交易
* 查看规则说明
* 放弃响应

自由行动不能无限触发游戏效果。

---

## 22. 出牌与响应结构

玩家执行攻击、技能或卡牌时，不应立即结算。

标准流程为：

```text
行动声明
→ 检查合法性
→ 支付费用
→ 锁定目标
→ 开启响应窗口
→ 玩家依次响应
→ 处理响应链
→ 结算原行动
→ 记录结果
```

### 22.1 行动声明

行动者必须公开：

* 使用什么行动
* 主要目标
* 基础费用
* 是否会开启响应

隐藏攻击只公开必要信息。

### 22.2 响应窗口

可能参与响应的玩家包括：

* 行动目标
* 与目标同阵营玩家
* 同区域玩家
* 拥有全局响应牌的玩家
* 特殊身份玩家

### 22.3 响应顺序

默认从目标开始，沿当前玩家顺序依次询问。

例如：

```text
玩家 A 攻击玩家 B
响应顺序：
玩家 B
→ 玩家 C
→ 玩家 D
→ 玩家 A
```

玩家 A 最后可以响应他人对自己行动的干预。

### 22.4 响应链

响应本身也可以被响应。

例如：

```text
A 使用攻击牌
B 使用闪避牌
C 使用禁止牌取消闪避
D 使用无效牌取消禁止
最终：
B 的闪避恢复生效
```

为了避免无限响应：

* 每个效果只能进入一次主要响应链
* 同一张卡不能重复参与
* 系统检测重复状态
* 达到响应深度上限后，由上帝强制结算

---

## 23. 遭受攻击时的特殊选择

公开规则整理中出现：

遭到攻击时，要么弃掉全部手牌，要么回答真心话。

该规则应当作为经典模式的重要特色保留，而不是普通彩蛋。

玩家在攻击即将造成伤害时，可以选择：

正常防御

继续使用护甲、闪避、技能或卡牌。

弃掉全部手牌

* 弃掉当前所有手牌
* 减免本次大部分或全部伤害
* 不能弃掉身份牌、种族牌和已装备物品
* 若没有手牌，不能选择此项

回答真心话

系统从游戏内问题库生成问题。

例如：

* 你是否拥有创界印记？
* 你是否曾在黑夜攻击过玩家？
* 你的身份是否属于魔王阵营？
* 你手中是否有复活牌？
* 你是否与某名玩家签订秘密契约？

玩家可以选择：

* 如实回答并获得较高减伤
* 拒绝回答并正常承受伤害
* 在特定身份允许下说谎，但承担被揭穿的惩罚

接受大冒险

某些规则允许用大冒险替代真心话。

大冒险必须是游戏内任务，例如：

* 下一回合不能进入城镇
* 必须向某玩家赠送一张牌
* 必须挑战一个怪物
* 必须在两轮内到达指定区域
* 必须公开一件装备
* 必须使用某种花色卡牌

不使用现实中的危险、侮辱或侵犯隐私任务。

---

## 24. 元宝发动真心话大冒险

网络规则整理普遍提到，玩家获得五枚元宝后，可以消耗元宝命令其他玩家进行真心话或大冒险。

经典模式暂定：

* 费用：`5` 枚元宝
* 时机：自己的行动阶段
* 目标：同区域玩家或满足卡牌范围的玩家
* 效果：目标选择真心话或大冒险

### 24.1 防止连续控制

同一玩家：

* 每轮最多被主动命令一次
* 完成大冒险期间不能再次获得相同任务
* 可以使用防御牌、身份能力或支付元宝拒绝
* 平安夜可能降低发动费用，但禁止恶意任务

### 24.2 完成奖励

目标若如实完成：

* 获得元宝
* 获得卡牌
* 获得声望
* 清除一项负面状态

发起者则可能获得：

* 情报
* 任务进度
* 身份奖励
* 社交控制效果

---

## 25. 交易阶段规则

玩家通常可以在自己的行动阶段交易。

交易内容包括：

* 基础卡牌
* 道具
* 装备
* 元宝
* 地产
* 情报
* 契约
* 礼物

不能直接交易：

* 隐藏身份牌
* 种族牌
* 已消耗的血牌
* 系统禁止转让的任务物品
* 未经规则许可的创界资格

交易流程：

```text
提出交易
→ 对方查看条件
→ 双方确认
→ 系统检查物品
→ 同时交换
→ 记录日志
```

系统必须防止一方先交付、另一方拒绝的情况。

口头承诺可以存在，但系统不保证履行。

---

## 26. 局部游戏插入规则

当玩家进入赌场、酒馆、迷宫或特殊格子时，可以插入局部玩法。

### 26.1 插入位置

局部游戏通常发生在：

```text
区域触发之后
→ 行动阶段之中
→ 玩家回合结束之前
```

### 26.2 局部游戏时限

局部游戏建议控制为：

* 单人挑战：`30` 至 `60` 秒
* 双人小游戏：`1` 至 `2` 分钟
* 多人小游戏：最多 `3` 分钟
* 大型副本：跨多个世界轮，但每轮只执行一小段

### 26.3 未参与玩家

未参与玩家可以：

* 观战
* 下注
* 使用允许的干扰牌
* 继续进行同步准备
* 在大型副本中等待自己的阶段

不能让其他玩家长时间没有任何可做的事情。

### 26.4 返回主回合

局部游戏结束后：

```text
发放奖励与惩罚
→ 修改主世界状态
→ 关闭局部界面
→ 返回行动阶段
→ 继续剩余行动
```

---

## 27. 平安夜特殊回合

游戏存在平安夜和礼物相关机制。

若世界牌触发平安夜，则标准黑夜阶段被替换。

```text
上帝宣布平安夜
→ 所有人选择礼物
→ 秘密指定赠送目标
→ 同时交换礼物
→ 依次开启礼物
→ 结算隐藏效果
→ 进入庆典行动阶段
```

### 27.1 平安夜限制

* 杀手不能发动普通暗杀
* 魔王不能直接造成伤害
* 玩家不能主动发起普通攻击
* 盗贼不能偷取礼物
* 允许恶作剧和诅咒礼物
* 允许交易、谈判和情报交换

### 27.2 礼物类型

包括：

* 元宝
* 道具
* 装备
* 卡牌
* 临时祝福
* 恶作剧
* 空盒
* 任务
* 身份线索

礼物可以匿名，也可以公开署名。

### 27.3 平安夜之后

平安夜并不停止主回合。

礼物结算后，玩家仍可执行公开行动，但攻击类行动受到限制。

---

## 28. 第 7 阶段：区域与世界行动

所有玩家完成公开回合后，世界中的非玩家实体开始行动。

执行顺序：

```text
区域持续效果
→ 怪物行动
→ NPC 行动
→ 商业结算
→ 任务推进
→ 地产结算
→ 阵营行动
→ 魔王势力推进
→ 神仙事件
→ 局部灾难
```

### 28.1 怪物行动

怪物根据：

* 仇恨
* 距离
* 区域
* 世界牌
* 魔王状态

选择目标。

### 28.2 地产与经济结算

处理：

* 租金
* 商店收益
* 贷款利息
* 股票波动
* 拍卖结果
* 产业维护费

### 28.3 任务推进

处理：

* 限时任务
* 大冒险
* 护送
* 区域防守
* 角色使命
* 阵营任务

---

## 29. 第 8 阶段：裂变与终局检查

每轮结束前，系统计算裂变变化。

### 29.1 裂变值增长来源

包括：

* 世界自然增长
* 玩家死亡
* 使用点卡改变规则
* 魔王行动
* 大型战斗
* 规则冲突
* 区域被占领
* 地产垄断
* 使用神器
* 世界牌特殊效果
* 完成高阶任务

### 29.2 裂变事件

达到阈值时，立即结算：

* 地图格损坏
* 路径断裂
* 区域规则改变
* 商店关闭
* 怪物强化
* 资源枯竭
* 身份暴露
* 创界山道路开启

### 29.3 创界山检查

系统检查：

* 是否达到开放轮数
* 是否有玩家拥有资格
* 是否完成关键任务
* 是否有阵营满足开启条件
* 是否出现创界山世界牌
* 是否达到最终裂变等级

满足条件后，下一轮进入创界阶段。

---

## 30. 第 9 阶段：轮次结束

最后处理：

```text
清除本轮状态
→ 弃置世界牌
→ 检查手牌上限
→ 结算回合末死亡
→ 更新日志
→ 保存当前状态
→ 将起始玩家标记传递
→ 进入下一轮
```

### 30.1 手牌上限

玩家默认手牌上限可以设为六张，以呼应初始六张手牌规则。

超过上限时：

* 主动弃牌
* 存入仓库
* 使用背包扩容
* 支付元宝保留
* 使用点卡临时扩大上限

### 30.2 自动保存

每轮结束后，服务器保存：

* 当前地图
* 玩家状态
* 手牌与牌库
* 隐藏身份
* 随机状态
* 行动日志
* 裂变值
* 当前阶段

用于掉线恢复和回放。

---

## 31. 裂变阶段的回合变化

进入裂变阶段后，在上帝判定前增加：

**裂变扩散阶段**

完整结构变为：

```text
轮次准备
→ 裂变扩散
→ 上帝判定
→ 世界状态
→ 秘密行动
→ 黎明
→ 公开行动
→ 世界行动
→ 裂变检查
→ 轮次结束
```

### 31.1 裂变扩散

每轮至少发生一项：

* 一组地图格失效
* 一条道路封锁
* 一个区域被改变
* 一个资源系统恶化
* 一个局部玩法关闭
* 创界山路径缩短
* 安全区域缩小

### 31.2 行动节奏加速

裂变阶段：

* 玩家行动时间缩短
* 地图移动增强
* 长期经营收益下降
* 创界资格获取速度提高
* 战斗危险增加
* 复活成本提高

目标是迫使玩家结束积累，前往创界山。

---

## 32. 创界阶段的回合变化

创界阶段不再使用完整普通地图回合。

回合变为：

```text
上帝翻终局牌
→ 创界山环境变化
→ 身份终局行动
→ 登山行动
→ 玩家响应与冲突
→ 守山者行动
→ 山体裂变
→ 创界资格检查
```

### 32.1 登山行动

玩家可以：

* 向上移动
* 挑战守卫
* 阻挡其他玩家
* 支援盟友
* 使用神器
* 完成身份任务
* 争夺规则钥匙
* 修改终局规则

### 32.2 身份公开

进入创界山后，部分身份会被强制公开。

仍需隐藏的身份可以保留至最终层。

### 32.3 最终轮限制

创界阶段应设置明确倒计时。

例如：

**创界山将在 5 轮后完全崩塌。**

倒计时结束后，无论玩家是否准备完毕，都必须进行最终判定。

---

## 33. 行动合法性检查

任何玩家行动都必须经过以下检查：

```text
是否轮到该玩家
→ 当前阶段是否允许
→ 玩家是否存活
→ 是否拥有所需卡牌或技能
→ 目标是否合法
→ 距离是否满足
→ 资源是否足够
→ 是否受到禁止状态
→ 模块是否启用
→ 是否存在规则冲突
```

失败时，系统必须说明原因。

例如：

```text
无法发动暗杀：
当前世界状态为平安夜，
主动暗杀行动被禁止。
```

不能只显示“操作失败”。

---

## 34. 规则冲突优先级

多种游戏规则同时存在时，采用以下优先级：

```text
上帝最终裁定
>
终局规则
>
世界牌规则
>
强制禁止效果
>
替代效果
>
免疫效果
>
身份规则
>
区域规则
>
职业技能
>
装备与卡牌修正
>
基础规则
```

### 34.1 禁止优先于允许

例如：

* 装备：本回合可以额外移动 `1` 次
* 世界状态：所有玩家禁止移动
* 结果：不能移动

### 34.2 特殊优先于一般

例如：

* 基础规则：死亡玩家退出游戏
* 点卡规则：死亡后可消耗点卡复活
* 结果：点卡作为特殊规则覆盖基础死亡规则

### 34.3 后发动不一定后结算

响应链按照规则优先级和栈结构结算，而不是简单比较时间。

### 34.4 上帝裁定必须记录理由

若出现无法自动判断的组合，上帝必须记录：

* 冲突规则
* 最终采用规则
* 优先级来源
* 状态变化

未来修正规则后，可以减少人工裁定。

---

## 35. 无限循环处理

可能出现：

```text
A 将伤害转给 B
B 将伤害转给 C
C 将伤害转回 A
```

系统应检测：

* 同一事件重复经过同一目标
* 同一规则重复触发
* 状态恢复至历史状态
* 响应链超过深度上限

检测到循环后：

1. 停止继续触发。
2. 回退到最近稳定状态。
3. 由上帝按照优先级结算。
4. 记录“规则循环中断”日志。
5. 必要时增加裂变值。

规则无限循环本身可以被视为世界不稳定的来源。

---

## 36. 联机等待与超时

### 36.1 普通玩家回合

建议默认：

* 思考时间：`60` 秒
* 最后提醒：`15` 秒
* 超时处理：自动结束剩余行动

### 36.2 响应窗口

建议默认：

* 普通响应：`10` 至 `15` 秒
* 复杂响应：`20` 秒
* 超时处理：默认不响应

### 36.3 秘密行动

超时后执行预设安全行为，不进行随机高风险操作。

### 36.4 局部游戏

超时视为：

* 弃权
* 采用默认选项
* 失去部分押注
* 由托管完成最低风险动作

### 36.5 暂停

好友房间可以由多数玩家投票暂停。

公开房间原则上不允许无限暂停。

---

## 37. 观战规则

观战者只能查看公开信息。

不能查看：

* 手牌
* 隐藏身份
* 黑夜选择
* 未公开牌库
* 私人交易
* 隐藏任务

拥有延迟观战模式时，可以在一定时间后查看完整信息。

真人上帝可以拥有更高观察权限，但其操作会被记录。

---

## 38. 回合日志

每轮生成两套日志。

### 38.1 玩家日志

显示玩家有权知道的内容。

例如：

* 第 `6` 轮：黑桃 `7`，世界进入黑夜。
* 昨夜玩家 B 受到袭击。
* 玩家 C 移动至万象赌场。
* 玩家 C 支付 `2` 枚元宝参与骰子对决。
* 玩家 C 获得创界印记。

### 38.2 完整裁定日志

仅用于回放、问题排查或游戏结束后公开。

例如：

* 杀手玩家 A 选择攻击玩家 B。
* 法师玩家 D 对玩家 B 使用护盾。
* 魔王效果使攻击伤害 `+1`。
* 长老玩家 C 封印魔王效果。
* 最终伤害为 `1`。

## 39. 标准世界轮示例

当前为第 `5` 轮。

上帝翻开：

**黑桃 8**

系统宣布：

世界进入长夜。
杀手、妖侠、法师、盗贼、魔王、神仙和长老可以行动。

黑夜阶段：

* 杀手选择攻击玩家 B
* 妖侠追踪玩家 C
* 法师保护玩家 B
* 盗贼尝试偷取玩家 D 的元宝
* 魔王诅咒黄金城
* 神仙赐福玩家 A
* 长老封印黄金城的一项诅咒效果

黎明：

* 玩家 B 昨夜受到袭击，但因未知力量保护，只失去 `1` 点生命
* 玩家 D 失去 `1` 枚元宝
* 黄金城受到轻度诅咒，本轮交易费用增加

公开讨论结束后，进入玩家行动。

玩家 A：

* 掷 `D20` 得到 `14`
* 换算为 `4` 点移动
* 移动到夜之村
* 触发身份调查事件
* 消耗一次主要行动调查玩家 C

玩家 B：

* 移动到万牌城
* 购买一张防御牌
* 向玩家 D 提出交易

玩家 C：

* 进入万象赌场
* 支付 `2` 枚元宝参与德州扑克式短局
* 获胜并得到一枚创界印记

玩家 D：

* 消耗 `5` 枚元宝，对玩家 A 发动真心话大冒险
* 玩家 A 选择大冒险：下一轮必须进入魔王领边界

世界结算：

* 黄金城诅咒生效
* 魔王势力增加
* 玩家 A 获得大冒险任务
* 本轮产生战斗、点卡规则干预和世界诅咒，裂变值增加 `3` 点

轮次结束，系统保存状态并进入第 `6` 轮。

## 41. 待核实的原剧细节

以下内容不能暂时视为完全确定的原剧规则：

1. 初始手牌是否准确为六张。
2. 世界牌是否直接使用基础手牌牌堆。
3. 黑桃之外三种花色的准确效果。
4. 黑夜唤醒顺序中的角色是否完全独立。
5. “神仙长老”是一个身份还是两个身份。
6. 妖侠、法师、盗贼、魔王、神仙和长老的准确任务。
7. 是否所有玩家每轮都掷一次二十面骰。
8. `D20` 点数是否直接等于移动距离。
9. “禁止说为什么”的准确惩罚。
10. 遭受攻击时是否只能在弃牌与真心话之间选择。
11. 真心话回答后是否完全免除攻击。
12. 五枚元宝发动大冒险的准确条件。
13. 完成真心话大冒险是否获得两枚元宝。
14. 神鬼契约的准确内容。
15. 隐藏攻击是否确实分为十二种。
16. 点卡与中途加入的完整规则。
17. 平安夜的礼物顺序和奖励方式。
18. 创界山是否采用“最先登顶直接获胜”。

这些内容在获得更可靠的剧集台词或画面依据后，再从“原作高度相关”升级为“剧中明确”。

## 42. 回合结构总结

《开天辟地之大地的裂变》的回合不能被简化为普通的：

```text
摸牌
→ 出牌
→ 结束
```

完整回合应当体现陆展博融合多种游戏的设计方式：

```text
上帝翻开世界牌
→ 扑克牌花色改变自然规则
→ 狼人杀式黑夜身份依次行动
→ 法师、杀手、妖侠、盗贼、魔王、神仙和长老互相影响
→ 黎明公开部分结果并进行身份推理
→ 玩家通过骰子在地图上移动
→ 大富翁式地产、房租和区域事件生效
→ 三国杀式卡牌响应与战斗展开
→ 真心话大冒险成为攻击和社交规则
→ 元宝用于交易和命令他人
→ 点卡允许玩家购买规则例外
→ 赌场、副本、酒馆等局部游戏插入回合
→ 所有规则冲突不断积累裂变值
→ 大地裂变改变下一轮结构
→ 最终所有回合转化为创界山争夺
```

这套结构既保留了原剧最有辨识度的规则，也让原剧没有完整交代的部分获得统一、稳定和适合联机的执行方式。

它的核心不是让每个玩家按照固定套路行动，而是让玩家每轮都面对一个由上帝重新定义的世界。
